- A `Qt6` reimplementation of the Electron based [Colorpicker](https://github.com/Toinane/colorpicker) by [Toinane](https://github.com/Toinane).
- An incomplete `Win32` implementation can be found inside the `/archived` directory. Abandoned as I no longer use Windows machines.
- The repository is MIT licensed except for the Qt based components inside `/qt6` directory, which are licensed under GPL 3.0, as required by Qt.
