A `Qt6` reincarnation of the Electron based [Colorpicker](https://github.com/Toinane/colorpicker) by [Toinane](https://github.com/Toinane).
An incomplete `Win32` implementation can be found inside the `archived` directory. Abandoned as I no longer use Windows machines.
